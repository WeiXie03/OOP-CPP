# Vowels

For this question, a **vowel** is defined to be one of the five lower case
letters *a*, *e*, *i*, *o*, or *u*.

In [enable1.txt](enable1.txt), how many words contain **all** five vowels
*exactly once*? The order of the vowels doesn't matter.

For example, `sequoia` and `education` both contain all five vowels exactly
once.


## Files

[enable1.txt](enable1.txt)

Write your answer in [challenge_3.cpp](challenge_3.cpp).


## Ideas (not for marks!)

- How many words in [enable1.txt](enable1.txt) contain all the vowels *exactly
  once* and in alphabetical order? What about reverse alphabetical order?
