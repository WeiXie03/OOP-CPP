# No Threes

What is the sum of the numbers in [numbers1.txt](numbers1.txt) that *don't*
contain the digit 3?

For example:

```
4
31
200
-9
20033
0
573
```

The sum of the numbers that don't contain the digit 3 is 4 + 200 + -9 + 0 =
195.


## Files

[numbers1.txt](numbers1.txt) contains one integer per line.

Write your answer in [challenge_1.cpp](challenge_1.cpp).


## Ideas (not for marks!)

- How would you modify your program to sum *only* numbers that contained the
  digit 3?
